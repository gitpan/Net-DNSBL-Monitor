These reports show the lookup results for the DNSBLs used to our IP space.

Source Data

The IP's used to generate this data are selected from our allocated IP space.

Methodology

The comparison and report generation is done using the Perl module Net::DNSBL::Monitor which provides a script that accepts one or more lists of IP addresses and can perform the DNSBL interrogations in parallel. In addition, it perform checks for missing PTR records and detects matches to GENERIC reverse host name patterns.
